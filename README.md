# command-line-notes-app

## About
***A simple app to take your notes***

## Technologies
* Javascript - Node.js

## Usage
#### Install Packages
    npm install

#### Add Note Example:
    node src/index.js add --title="title" --body="body"

#### Remove Note Example
    node src/index.js remove --title="title"

#### List Note Example
    node src/index.js list

#### Read Note Example
    node src/index.js read --title="title"
